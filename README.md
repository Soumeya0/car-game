# 🚗 Highway Racer

A highway racing game built with Python and Pygame. Use arrow keys to switch lanes and avoid oncoming cars.

## How to Play

- Use **LEFT** and **RIGHT** arrows to change lanes
- Avoid enemy cars
- Each car you pass = 10 points
- Speed increases as your score gets higher

## Requirements

- Python 3.x
- Pygame

## Installation

```bash
pip  or pip3 install pygame
python or python3 racing_game.py
